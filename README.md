## [Components](https://components-ui.davidcamelo.com) and [Storybook](https://storybook.davidcamelo.com) UI
[![Docker Image CI / Kubernetes Deployment CD](https://github.com/DavidCamelo/components-ui/actions/workflows/deploy.yml/badge.svg)](https://github.com/DavidCamelo/components-ui/actions/workflows/deploy.yml)

This repository contains a collection of reusable UI components built with React and styled using Tailwind CSS. The components are documented and showcased using Storybook, providing an interactive environment for developers to explore and test the components.
